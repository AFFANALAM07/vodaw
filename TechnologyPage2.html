India's Import Electronic Commodity (2021-2022)

The visualization shows India's import and re-import of electronic commodities for the fiscal year 2021-2022. It presents the total import value of various electronic products, with items like mobile phones, SIM cards, and other telecommunications equipment contributing significantly to the overall import value of ₹122,356.31 crore.A slicer has been added to filter the data by specific commodities, allowing users to explore detailed import figures for individual electronic goods.
